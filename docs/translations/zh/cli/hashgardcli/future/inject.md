# hashgardcli  future inject

## 描述
对盒子进行通证注入。



## 用法
```shell
hashgardcli future inject [box-id] [amount]  --from
```



### 子命令

| 名称   | 类型   | 必需 | 默认值 | 描述         |
| ------ | ------ | -------- | ------ | ------------ |
| box-id | string | 是       |        | 盒子的 id |
| amount | int   | 是       |        | 存款的数量   |



## Flags

**全局 flags、查询命令 flags** 参考:[hashgardcli](../README.md)

## 例子
### 进行存款

```shell
hashgardcli future  inject boxac3jlxpt2ps 1800 --from
```
