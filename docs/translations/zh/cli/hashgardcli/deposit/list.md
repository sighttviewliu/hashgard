# hashgardcli deposit list

## 描述
查询指定类型盒子列表

## 用法
```shell
hashgardcli deposit list
```

### flag

**全局 flags、查询命令 flags** 参考:[hashgardcli](../README.md)


## 例子
### 查询盒子信息

```shell
hashgardcli deposit list
```
