# hashgardcli  deposit inject

## 描述
对盒子进行通证注入。



## 用法
```shell
hashgardcli deposit inject [box-id] [amount] [flags]
```


### 子命令

| 名称   | 类型   | 必需 | 默认值 | 描述         |
| ------ | ------ | -------- | ------ | ------------ |
| box-id | string | 是       |        | 盒子的 id |
| amount | int   | 是       |        | 存款的数量   |


## Flags

**全局 flags、查询命令 flags** 参考:[hashgardcli](../README.md)

## 例子
### 进行存款

```shell
 hashgardcli deposit inject boxab3jlxpt2pt 10000 --from
```
