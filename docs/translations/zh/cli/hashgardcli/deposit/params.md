# hashgardcli deposit params

## 描述
关于 deposit 的参数设定

## 用法


```shell
hashgardcli deposit params
```
## Flags
**全局 flags, 查询命令 flags** [hashgardcli](../README.md)

## 例子

```shell
hashgardcli deposit params
```

成功后，返回信息：

```shell
Params:
  DepositBoxCreateFee:			1000000000000000000000agard
  DisableFeatureFee:			1000000000000000000000agard
  DescribeFee:			100000000000000000000agard
```
