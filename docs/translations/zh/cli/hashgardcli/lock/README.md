# hashgardcli lock

## 描述
用户对自己的资产进行有限期的锁定

## 用法

```shell
hashgardcli lock [subcommand]
```

打印子命令和参数

```shell
hashgardcli lock --help
```
## 子命令
| 名称                        | 功能     |
| -------------------------- | ------------ |
| [params](params.md)        | 参数及费用    |
| [create](create.md)        | 创建锁仓     |
| [describe](describe.md)   |  对锁仓进行描述 |
| [query](query.md)  |  查询指定 id 锁仓 |
| [list](list.md)   | 查询所有锁仓列表  |
| [search](search.md)  | 按锁仓名字进行查询  |
