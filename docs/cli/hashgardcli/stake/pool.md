# hashgardcli stake pool

## Description

Query values for amounts stored in the staking pool:

## Usage

```shell
hashgardcli stake pool [flags]
```

## Flags

**Global flags, query command flags** [hashgardcli](../README.md)

## Example


```shell
hashgardcli stake pool --trust-node
```
