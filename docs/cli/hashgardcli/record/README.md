# hashgardcli record

## Description

record subcommand

## Usage

```shell
 hashgardcli record [subcommand]
```

## subcommand

| Name                           | Description                    |
| ------------------------------- | -------------------------- |
| [create](create.md) | create record       |
| [query](query.md)             | query record based on specified hash   |
| [list](list.md)             | query all records|
