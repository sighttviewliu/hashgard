# hashgardcli box list-box

## Description
Query box list

## Usage
```shell
hashgardcli box list-box [flag]
```

### flag

| Name    | Description   |
| ------- | -------- |
| lock    | lock box |
| deposit | deposit box|
| future  | future box  |


**Global flags, query command flags** [hashgardcli](../README.md)

## Example
### Query box list

```shell
hashgardcli box list-box future
```
