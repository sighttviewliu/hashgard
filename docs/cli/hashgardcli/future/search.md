# hashgardcli future search

## Description
Search future Box by Name 
## Usage
```shell
hashgardcli future search [name]
```
## Flags

**Global flags, query command flags** [hashgardcli](../README.md)

## Example
### 搜索
```shell
hashgardcli future search pay
```
