# hashgardcli future query

## Description
Query by future box ID 

## Usage
```shell
hashgardcli future query [box-id]
```

## Subcommands

| Name| Type  | Required | Default   | Description        |
| ------ | ------ | -------- | ------ | ------------ |
| box-id | string | true       |        | 盒子的 id |



## Flags

**Global flags, query command flags** [hashgardcli](../README.md)
